### 페이지 단위 작성

`<DocPage>` 블록 하나가 A4 한 페이지로 렌더링됩니다.

```mdx
<DocPage>

## 제목

본문 내용을 작성합니다.

</DocPage>
```

:::note
h2 제목에 순서 번호를 붙이지 않습니다. 목차가 h2를 자동 수집하면서 챕터 인덱스를 부여합니다.
:::

특수 타입으로 표지와 목차 페이지를 만들 수 있습니다.

| 타입                     | 용도        | 설명                            |
| ------------------------ | ----------- | ------------------------------- |
| `<DocPage>`              | 일반 콘텐츠 | 본문, 차트, 코드 등             |
| `<DocPage type="cover">` | 표지        | 제목, 부제, 작성자              |
| `<DocPage type="toc">`   | 목차        | h2/h3를 자동 수집해서 목차 생성 |

:::note
목차는 문서 내 모든 `.doc` 요소에서 h2/h3를 수집합니다. 위치와 무관하게 작동하며, 표지의 `##`는 내부적으로 `<p>` 태그로 렌더링되므로 목차에 포함되지 않습니다.
:::

### 스타일 커스터마이징

`theme.tsx`가 있으면 해당 콘텐츠에만 커스텀 MDX 컴포넌트가 적용됩니다.
없으면 기본 스타일이 사용됩니다.

```ts
// src/content/my-book/theme.tsx
import { docMDXComponents } from '../../../components/layout/DocMDXComponents'

export const mdxComponents = {
  ...docMDXComponents,
  h2: ({ children }) => <h2 style={{ color: '#7c3aed' }}>{children}</h2>,
}
```

:::tip
기본 컴포넌트를 스프레드한 뒤 원하는 태그만 오버라이드하면 나머지는 기본 스타일이 유지됩니다.
:::

### PDF 출력

1. 개발 서버를 실행하고 문서를 선택합니다.
2. 우측 하단의 PDF 저장 버튼을 클릭합니다.
3. 인쇄 대화상자에서 다음을 설정합니다:
   - 대상: PDF로 저장
   - 여백: 없음
   - 배경 그래픽: 체크

:::warning
같은 DocPage 안에서 콘텐츠가 A4를 넘어가면 수동으로 페이지 브레이크를 삽입하세요: `<div className="page-break-before" />`
:::
