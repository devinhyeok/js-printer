### 카탈로그 등록

`catalog.ts`에 등록해야 선택 목록에 표시됩니다. `id`는 콘텐츠 폴더명과 일치해야 합니다.

```ts
// src/content/document/catalog.ts
import type { CatalogItem } from '../catalog-types'

export const catalog: CatalogItem[] = [
  // 폴더 없이 등록하면 최상단에 바로 표시
  { id: 'my-book', label: '내 문서' },

  // 폴더로 그룹핑하면 접을 수 있는 트리 구조
  {
    folder: '예제',
    children: [
      { id: 'demo-doc', label: 'JS Printer 설명서' },
    ],
  },
]
```

- `id`는 콘텐츠 폴더명과 일치해야 합니다.
- `label`은 선택 목록에 표시될 이름입니다 (생략 시 `id`가 표시됨).
- 배열 순서가 곧 표시 순서입니다.

카탈로그 타입 정의는 다음과 같습니다.

```ts
// catalog-types.ts
interface BookItem {
  id: string
  label?: string
}
interface FolderItem {
  folder: string
  children: BookItem[]
}
type CatalogItem = BookItem | FolderItem
```

:::note
프레젠테이션도 동일하게 `src/content/presentation/catalog.ts`에 등록합니다.
:::
