### 헤딩 규칙

| 레벨    | 역할             | TOC 추적 |
| ------- | ---------------- | -------- |
| **h2**  | 챕터 제목        | O        |
| **h3**  | 슬라이드 제목    | O        |
| **h4+** | 콘텐츠 내부 라벨 | X        |

- 슬라이드당 h2 **또는** h3 하나만 사용
- h2 → TOC에서 챕터 구분 (굵게, 구분선)
- h3 → TOC에서 하위 항목 (들여쓰기)
- h4 → 컬럼 라벨, 설명 등 시각적 용도
