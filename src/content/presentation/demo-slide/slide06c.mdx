### 컬럼 분할 문법

`---` 하나로 콘텐츠를 자동 분할합니다.

| 타입        | 비율    | 특징           |
| ----------- | ------- | -------------- |
| two-content | 50 : 50 | 균등 2단       |
| comparison  | 50 : 50 | h4 라벨 강조   |
| caption     | 35 : 65 | 비대칭, 캡션용 |

- 제목(h2/h3)은 자동으로 컬럼 상단에 배치
- `---` 없으면 단일 컬럼으로 렌더링
- 별도 컴포넌트 없이 마크다운만으로 완성
