### 타입 시스템

#### 자동 스타일링

`type` 하나로 배경, 색상, 타이포가 자동 적용됩니다.

---

| 타입        | 용도          |
| ----------- | ------------- |
| cover       | 표지          |
| content     | 일반 콘텐츠   |
| section     | 챕터 구분     |
| two-content | 좌우 2단      |
| comparison  | 좌우 비교     |
| caption     | 캡션 + 콘텐츠 |
