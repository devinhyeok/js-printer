### 페이지 단위 작성

`<div className="page">` 블록 하나가 A4 한 페이지로 렌더링됩니다.

```mdx
<div className="page">

## 1장. 제목

본문 내용을 작성합니다.

</div>
```

:::note
`page` 클래스 안에서만 A4 크기와 그림자, 페이지 번호가 적용됩니다.
:::
